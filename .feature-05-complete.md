# Feature 5: Git-Diff Style Suggestions — ✅ COMPLETE

**Implementation Date:** 2026-02-08  
**Status:** Deployed and running

## Summary

When Kona suggests changes to script/description/voiceover/thumbnails, they render as inline diffs with Accept/Reject buttons. This makes it easy for Nora to review and apply suggestions with one click.

## Components

### 1. System Prompt (server.js)
- Added SUGGEST format instructions to `buildSystemPrompt()` and `buildChannelSystemPrompt()`
- Format: `<<<SUGGEST tab="X" section="Y">>> ---OLD--- ... ---NEW--- ... <<<END_SUGGEST>>>`
- Supports: script, description, voiceover, thumbnails tabs

### 2. Frontend Parser (app.js)
- `processSuggestions()` — Regex parser for SUGGEST blocks
- Generates unique IDs for each suggestion
- Renders diff UI with header, old/new text, and action buttons
- Called from `processMessageContent()` for assistant messages only

### 3. User Actions (app.js)
- `acceptSuggestion(id, tab, oldText, newText)` — Finds text in CodeMirror editor, replaces it, saves video
  - Shows "✓ Applied" on success
  - Shows "⚠ Text not found" if content changed
- `rejectSuggestion(id)` — Marks as "✗ Dismissed" (visual only, no side effects)

### 4. Styling (style.css)
- `.suggestion-diff` — Container with border and rounded corners
- `.diff-header` — Tab name, section, and action buttons
- `.diff-body` — Split view with OLD (red bg) and NEW (green bg)
- `.diff-accept` / `.diff-reject` — Styled buttons
- `.suggestion-accepted` / `.suggestion-rejected` — Opacity states after action
- `.diff-status` — Status labels (accepted/rejected/warn)

## Usage

### For Kona (AI):
When suggesting changes, use this exact format:
```
<<<SUGGEST tab="script" section="Opening Hook">>>
---OLD---
exact text to replace
---NEW---
replacement text
<<<END_SUGGEST>>>
```

### For Nora (User):
1. Ask Kona for suggestions
2. Review the rendered diff
3. Click ✓ Accept to apply, or ✕ Reject to dismiss

## Technical Details

- **Text Matching:** Uses exact string replacement (whitespace-sensitive)
- **Editor Integration:** Works with CodeMirror editors for all tabs
- **Auto-save:** Triggers `saveVideo()` after accepting a suggestion
- **Multiple Suggestions:** Can render multiple diffs in one message
- **Safety:** Shows warning if OLD text not found (prevents accidental replacements)
