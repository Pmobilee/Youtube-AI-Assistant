## What It Does

When Kona suggests changes to video scripts, descriptions, voiceover notes, or thumbnail ideas, they now render as beautiful inline diffs with Accept/Reject buttons — just like GitHub code reviews.

## How Kona Uses It

In any chat (Script, Description, or Thumbnail), Kona can suggest changes using this format:

```
<<<SUGGEST tab="script" section="Opening Hook">>>
---OLD---
Welcome to my channel!
---NEW---
Hey everyone! Welcome back to the channel!
<<<END_SUGGEST>>>
```

### Parameters:
- **tab**: `script`, `description`, `voiceover`, or `thumbnails`
- **section**: (optional) Name of the section being changed for context

### Multiple suggestions:
Kona can include multiple SUGGEST blocks in one message to propose several changes at once.

## How Nora Uses It

When a suggestion appears:

1. **Review the diff** — Red shows what will be removed, green shows what will be added
2. **Click ✓ Accept** — Instantly applies the change to the editor and saves
3. **Click ✕ Reject** — Dismisses the suggestion
4. **States after action:**
   - ✓ Applied — Change was successfully made
   - ✗ Dismissed — Suggestion was rejected
   - ⚠ Text not found — The old text no longer exists (maybe already edited)

## Technical Details

### Implementation:
- **Backend:** System prompt in `buildSystemPrompt()` and `buildChannelSystemPrompt()` (server.js)
- **Frontend:** `processSuggestions()` parser with Accept/Reject handlers (app.js)
- **Styling:** `.suggestion-diff` classes with light/dark theme support (style.css)

### How Accept Works:
1. Finds the editor for the specified tab
2. Searches for exact match of OLD text
3. Replaces with NEW text
4. Auto-saves the video
5. Updates button to "✓ Applied"

### Smart Matching:
- Uses `.trim()` on both old and new text for flexible matching
- If text not found, shows warning instead of failing silently
- Handles special characters and multi-line text correctly

## Usage Tips for Kona

**Good SUGGEST formatting:**
```
<<<SUGGEST tab="script" section="Intro">>>
---OLD---
Today we're going to talk about AI.
---NEW---
In this video, I'll show you how AI is changing everything.
<<<END_SUGGEST>>>
```

**What makes it work:**
- Copy the EXACT text from the current script (spacing, line breaks, everything)
- Keep suggestions focused (one logical change per block)
- Use section names to give Nora context
- Multiple small suggestions > one giant suggestion

**Common mistakes to avoid:**
- Paraphrasing the OLD text (must match exactly)
- Including surrounding text that shouldn't change
- Forgetting to specify the correct tab
